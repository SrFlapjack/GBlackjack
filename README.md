# GBlackjack

Standalone blackjack for Garry's Mod.

## Features

- Spawnable GBlackjack table.
- Solo, multiplayer, and AI bot play.
- Configurable blackjack rules and payouts.
- Dealer messages, bot personas, wagers, hit, stand, and double down.
- Source-friendly Lua for server owners who want to customize behavior.
- Optional nMoney2 Wallet wager support when nMoney2 is installed.
